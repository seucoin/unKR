Install
=======

Create environment
------------------

Create a virtual environment directly (recommend anaconda)

.. code-block:: bash

    conda create -n unKR python=3.8
    conda activate unKR

Install from source
-------------------

.. code-block:: bash

    git clone https://github.com/CodeSlogan/unKR.git
    cd unKR
    python setup.py install

Install by pypi
---------------

.. code-block:: bash

    pip install unKR
